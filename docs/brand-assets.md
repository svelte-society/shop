# Brand assets

The storefront brand files are pinned to a single snapshot of the current Svelte Society website repository so their provenance is reproducible.

- Repository: [`svelte-society/sveltesociety.dev`](https://github.com/svelte-society/sveltesociety.dev)
- Default branch at retrieval: `main`
- Commit: [`14d31760a741eb6413b5df22538268939357288c`](https://github.com/svelte-society/sveltesociety.dev/commit/14d31760a741eb6413b5df22538268939357288c)

| Local file | Upstream path |
| --- | --- |
| `static/brand/svelte-society.svg` | `src/routes/(app)/society-logo.svg` |
| `static/fonts/manrope-variable.woff2` | `static/fonts/Manrope.woff2` |

The SVG is the Society-owned identity mark identified by the approved project brief. It is used here for the official Society storefront; copying it does not grant a separate trademark license.

Manrope is distributed under the [SIL Open Font License 1.1](https://github.com/davelab6/manrope/blob/master/license.txt). The Society repository does not keep a separate license file beside its font binary. The WOFF2 file was copied byte-for-byte and only renamed locally for a stable, descriptive URL.
